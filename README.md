## Features

- Đăng nhập/ Đăng Ký/ Đăng xuất
- Xem danh sách các sản phẩm
- Xem chi tiết sản phẩm
- Thêm vào giỏ hàng/ Xóa khỏi giỏ hàng
- Đặt hàng
- Xem đơn hàng trên database
